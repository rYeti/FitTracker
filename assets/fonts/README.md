# ForgeForm Fonts

To use the ForgeForm branding, download the following font files from Google Fonts and place them in this directory:

## Montserrat

Visit: https://fonts.google.com/specimen/Montserrat

Download and place these files here:
- `Montserrat-Regular.ttf`
- `Montserrat-Medium.ttf`
- `Montserrat-SemiBold.ttf`
- `Montserrat-Bold.ttf`

## Exo 2

Visit: https://fonts.google.com/specimen/Exo+2

Download and place these files here:
- `Exo2-Regular.ttf`
- `Exo2-Medium.ttf`
- `Exo2-SemiBold.ttf`
- `Exo2-Bold.ttf`

## Quick Download Instructions

1. Visit each Google Fonts link above
2. Click the "Download family" button
3. Extract the ZIP file
4. Copy the TTF files listed above to this `assets/fonts/` directory
5. Run `flutter pub get` to recognize the new assets
6. Restart your app

## Alternative: Use System Fonts

If you don't want to download custom fonts, the app will fall back to the system's default sans-serif font.
